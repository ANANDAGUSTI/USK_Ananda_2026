🧾 Aplikasi Kasir Sederhana (PHP)

Aplikasi Kasir berbasis PHP ini dibuat untuk membantu proses transaksi penjualan secara sederhana, mulai dari pengelolaan produk hingga pencatatan transaksi oleh kasir.

Fitur Utama
Authentication
- Login pengguna (Admin & Petugas)
- Logout sistem
- Session management

Admin
- Kelola data produk (CRUD)
  - Tambah produk
  - Edit produk
  - Hapus produk
  - Lihat daftar produk

Petugas (Kasir)
- Melihat daftar produk
- Menambahkan produk ke keranjang
- Mengelola keranjang (update & hapus item)
- Melakukan transaksi (checkout)

Transaksi
- Perhitungan total otomatis
- Penyimpanan data transaksi ke database
- Pengurangan stok barang otomatis

Teknologi yang Digunakan

- PHP (Native / Procedural)
- MySQL / MariaDB
- CSS
